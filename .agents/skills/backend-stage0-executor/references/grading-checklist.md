# Grading Checklist

- endpoint is publicly reachable
- query handling is correct
- real Genderize call is made
- `gender`, `probability`, and `sample_size` are correct
- `is_confident` logic is correct
- missing and invalid inputs return the correct error envelope
- no-prediction edge cases return the exact message
- live CORS header is present
- repeated requests do not take the server down
- README and submission data match the live deployment
